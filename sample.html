Fresh Graduate | PHP & MEAN Stack Developer | Health IT Innovator

About Me

I'm a fresh graduate with a passion for developing health-focused IT solutions. I specialize in both PHP (CodeIgniter, MySQL, Apache) and MEAN stack (MongoDB, Express.js, Angular, Node.js), and have hands-on experience building real-world applications to improve public health operations.

Projects

PhilHealth Ticketing System

A full-stack web application using CodeIgniter, Apache, PHP, and MySQL for managing ticket requests from health facilities. Features include validation, document upload, automated transaction numbers, and admin status updates.

MIHIS - Malnutrition & Immunization Health Info System

A mobile-friendly health information system designed using the MEAN stack to streamline malnutrition monitoring and immunization tracking. Integrated with SMS and Gmail notifications for mothers, e-vaccination card generation, analytics dashboards, and scheduling tools for midwives and barangay health workers.

Skills

  • Languages: PHP, JavaScript, SQL, TypeScript
  • Frameworks: CodeIgniter, Angular, Express.js, Tailwind CSS
  • Stacks: LAMP (Linux, Apache, MySQL, PHP), MEAN (MongoDB, Express, Angular, Node)
  • Tools: Git, Gmail API, SMS Gateway, MongoDB Compass
  • Soft Skills: Analytical thinking, Team collaboration, Project documentation
